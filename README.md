This BrightLearn research assignment is a beginner level worksheet covering foundational topics in data analytics, AI, and data science. It includes 37 questions on databases, data warehouses/lakes/marts, structured vs. unstructured data, ETL, machine learning, deep learning, NLP, LLMs, APIs, JSON, cloud computing, Python, Jupyter Notebooks, data ethics, privacy (GDPR), bias, and data-related job roles. Students must answer all questions by hand in full sentences, include diagrams or examples for bonus marks, cite sources, and submit scanned copies via email. The assignment ends with personal reflection questions about the student’s learning experience and the future impact of AI and data.
